Managed

  • Annotation

Annotation that is managed by some domain.

When used for any Class, it means that the instance of that Class is managed by the lifestyle specified by #value().

When used for any field, it means that the field is managed as Property. Specifying a Lifestyle is meaningless.

name()String

String

Configure the name of the associated property. Default value is empty.

value()Class?Lifestyle

Class?Lifestyle

Configure the lifestyle of the instance which is annotated class. The default value is I#prototype(Class).
